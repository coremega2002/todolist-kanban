# todolist-kanban

A React application created with Codiux.

## Getting Started

Install dependencies:

```bash
pnpm install
```

Run development server:

```bash
pnpm run dev
```

Build for production:

```bash
pnpm run build
```

## Framework

- React 18.3.1
- vite bundler
- TypeScript enabled

## Features

- Hot Module Replacement
- TypeScript support
- ESLint configuration
- Production-ready build
